Project is still pre-alpha:
 Get started Next Goals

Featuring

  • (social) Authentication & Login
  • Blazingly Fast Single-Page-Web UI
  • Mobile-friendly Infinite Scrolling
  • Archetype-Oriented Database (using JSONB)
  • JSON-API compatible Rest-API
  • Automatic API-Limits
  • Fully Supported (optional) Modules:
    • Likes
    • Comments
    • Private Messaging

Build using

  • Flask, SQLAlchemy on Python3
  • React in the Redux-Pattern
  • Postgresql & JSONB
  • Semantic Styles with SASS & Bourbon
  • true separation of concerns
